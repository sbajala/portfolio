Overview

This final project was done in teams of two and I worked on the server and the Album component. The goal of this application was to create a React application that communicates with the server-side NodeJS via a REST interface and use Discogs API. In summary, users are able to create new playlists, search songs by artist or album, save a song to a playlist, view their playlists, and remove a song from their playlist.



When I search "Paramore", I can see all of the songs that I can add to my playlist.



When I click on the + button, I can name and create a new playlist to save the song to.
