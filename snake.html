Snake


View on Github

Overview

This is a C++ final project done in teams of two. We had to recreate a console version of the classic arcade game Snake. I worked on creating the Person, Apple, and on the game logic (winning or losing).



When the player successfully eats an apple, the score and lenght increases by one.



When the player touches the wall boundaries (or itself), the game is over and they are asked if they want to play again.
