Overview

The goal of this project was to demonstrate our knowledge and skills in CRUD operations, UI design using XAML, validation, and data persistence. For this project, I worked on the login and sign up feature. I equally worked on the User class and created the database in which users would be stored in.



When users sign in, they are redirected to the main page. If they click on "add to cart" for an item, the details are shown to the right.



When users press "add to cart" from the gray dialog box, the item is added to the card. Users can then see all of the items in their cart.
