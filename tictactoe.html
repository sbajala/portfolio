Overview

This final project was done in teams of two and I worked on the views and ControllerPanel. The goal of this project was to practice the MVC pattern using Java Swing graphical user interface. Looking back at my code, I could see that the creation of the buttons and event listeners could have been done with the help of a loop instead of having duplicate code.



Player X starts. Users can view their score, change board size, or play again.



When a player wins, their score is updated and the message indicates the winner. Grid is disabled, so users can see their winning line and decide if they want to play again.
