Overview

This was the first project we had to do and at this point of our studies, we haven't yet seen databases or classes. The goal of this project was to create a C++ console application that simulates a bank in which we would practice using variables, creating functions, using loops and arrays, and thinking about the flow of the program. In summary, users are able to add or remove a bank account, deposit or withdraw from their accounts, view bank accounts by account number or balance as well as view other information.
